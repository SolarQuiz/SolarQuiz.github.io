Sobre SolarQuiz.

SolarQuiz é um quiz simples para TODAS as idades (De preferência, para o Fundamental 2.), criado como um Trabalho na escola do criador. (A próposito, sou eu :D).

Sendo assim, SolarQuiz é simples, intuitivo, é criado com o próposito de aprendizado, não espere muito muito dele..

Se você é um programador (como eu), você deveria dar uma olhada no Source Code (Codigo Fonte) desse jogo, você vai gostar. (Ou não, por que é extremamente mal programado)

A redestribuição do binario desse jogo ou o codigo fonte do mesmo com fins lucrativos ou não, é expressamente proibido, assim que é permitido o fork do Codigo Fonte para sugestão de Pull-Requests ou do mesmo.
